Review Bot is a web scraper built through Python scripts that use the Selenium ChromeDriver library to automate web scraping through a headless (run in memory) instance of Google Chrome. The data from the latest 20 reviews from each specified URL is then compiled into a CSV file and uploaded to an Amazon S3 Storage Bucket. The bot runs every 10 minutes on a Heroku web server, and this front-end display of data from the S3 instance is hosted by Github Pages. All of this is run at zero cost.
